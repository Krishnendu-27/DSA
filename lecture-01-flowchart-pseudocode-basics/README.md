## Flowchart Shapes

- **Oval (Terminator)**
  - Used for **Start / End**

```
 ( Start / End )
```

- **Parallelogram**
  - Used for **Input / Output**

```
  / Input Output /
```

- **Rectangle**
  - Used for **Processing / Operation**

```
 [ Process ]
```

- **Diamond**
  - Used for **Decision (Condition check)**

```
   < Decision >
```

## Print Count from 1 to N

- When we don’t know N, we take it as input

![flowchart-of-printing-till-N-numeber](../assets/img-6-flowchart-of-printing-till-N-numeber.png)

### Pseudocode

```
start
take n input
create variable i = 1
while i <= n
  print i
  increase i
end
```
