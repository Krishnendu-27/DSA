## Features of C++

- **statically typed**: we have to define the variable type before using it (we know what type it is).

- **speed**: it is close to low-level + compiled, that’s why it is very fast (faster than many high-level languages, but not faster than assembly).

- **platform independent (partially)**: C++ source code is portable, but compiled binary is platform dependent.

## Creator of C++

- **Bjarne Stroustrup** is the creator of C++.
- its history / landing story is very interesting.

---

## Story behind C++

- Bjarne did not initially want to create C++.

- he liked OOP concept from **Simula** language.

- during his PhD days, he tried combining OOP + Simula, but it failed.

- then he tried the same concept in C.

- he created a preprocessor called **Cfront (C pre)**.

- Cfront:
  - took C code with OOP features
  - converted it into pure C code
  - then compiled it

- benefit:
  - no performance loss
  - OOP support added to C

- this later became **C++**.

## Development of C++

C++ is still actively developed.

- **C++11** → major update
  - introduced `auto`, lambda functions, smart pointers

- **C++14** → improvements (generics, etc.)

- **C++17, C++20, C++23** → modern features, some styles inspired by languages like Rust
